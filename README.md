# MGuard
MGuard web/app flutter

#### `Flutter` version: 3.16.1
#### `Dart` version: 3.2.1

## 필독 사항
아래 문서들을 프로젝트를 시작하기전에 꼭 읽어주세요.

# How to start
1. 프로젝트 터미널에 `dart run build_runner build --delete-conflicting-outputs` 로 generated 파일 생성
2. 타겟을 앱(실기기 권장)으로 main.dart 실행 시 TPASS 앱 실행
3. 타겟을 웹(chrome)으로 main.dart 실행 시 관리자 웹 실행
