# MGuard Logic
MGuard 프로젝트의 로직에 대한 문서입니다.

## 구성요소
- 관리자 페이지
- 사용자 앱

## 특이사항
- 삼성 녹스 사용
- iOS Device Configuration Profile 사용

## 포함된 기능
- 사용자의 핸드폰에 카메라를 제어하는 기능
- 사용자의 핸드폰 카메라 상태를 체크하는 기능
- 비콘 탐색
- NFC 태깅
- 위치 체크 기능

## 로직 플로우
1. 사용자의 앱 설치
2. 권한 및 약관 동의 (+ 삼성 녹스)
3. 비콘 / NFC / 수동 으로 사용자의 핸드폰 카메라를 차단
4. 위치 / 수동 으로 사용자의 핸드폰 카메라를 허용

## 로직 플로우 (관리자)
1. 관리자 페이지 접속
2. 관리자 페이지에서 방문객의 정보를 조회

## 로직별 폴더 매핑
- 권한/약관 체크 및 동의
  - /lib/features/permission
- 가이드 정보
  - /lib/features/guide
- 메인 페이지 (카메라 제어, 비콘, NFC 등)
  - /lib/features/home
- 위치 체크
  - /lib/features/location
